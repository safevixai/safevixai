# GeminiProvider Module - AI Chatbot Service

## Overview

The `GeminiProvider` module provides an interface for interacting with the Google Gemini 1.5 Flash large language model within the SafeVixAI chatbot service. It translates OpenAI-style message formats into the Gemini-specific format and handles API requests to generate text responses. This module enables the platform to leverage Gemini's large context window for handling complex queries and long conversations related to road safety.

## Architecture

This module is part of the AI Chatbot Service within the SafeVixAI platform. It acts as a provider, abstracting the specifics of interacting with the Gemini API. It receives requests from the `chatbot_service` backend, formats the input, sends the request to the Gemini API, and returns the generated text. The `GeminiProvider` is one of several providers used by the system to access different LLMs.
