# Drug Information Tool

## Overview

The `drug_info.py` module provides a tool for the SafeVixAI chatbot to retrieve information about medications from the Open FDA API. This tool is crucial for the first-aid chatbot context, enabling it to provide relevant information about drug indications, warnings, dosage, and contraindications.

## Architecture

This module is part of the AI Chatbot Service within the SafeVixAI platform. It is a utility tool designed to be called by the chatbot logic to answer user queries about medications. It interacts with the external Open FDA API to fetch drug information. The module is used by the FastAPI backend and accessed through the Next.js frontend via the chatbot interface.
