# Challan Tool Module

## Overview

The `ChallanTool` module provides functionality for calculating traffic challan amounts based on violation codes, vehicle classes, state codes, and repeat offender status. It integrates with the backend API to determine the fine amount and offers an `infer_and_calculate` method to extract relevant information from user messages and client IP addresses. This module is a core component of the AI Chatbot Service, enabling users to quickly estimate challan costs.

## Architecture

This module resides within the `chatbot_service` and is part of the AI Chatbot Service, which interacts with the user via the Next.js frontend. It utilizes a FastAPI backend for challan calculations and leverages the `BackendToolClient` to communicate with the API. The module uses a local embedding function for state detection. The module interacts with the Supabase Auth system for user authentication, although no specific auth details are used within this module.
