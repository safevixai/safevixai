# Emergency Tool Module

## Overview

The `EmergencyTool` module provides an interface for the AI chatbot to interact with the backend service to retrieve information about nearby emergency services. It leverages a `BackendToolClient` to communicate with the FastAPI backend and fetch relevant data based on geographical coordinates. This module is crucial for enabling the chatbot to assist users in emergency situations by providing access to nearby resources.

## Architecture

This module resides within the `chatbot_service\tools` directory and is a component of the AI Chatbot Service. It acts as an intermediary between the chatbot's logic and the backend API, specifically the `/api/v1/emergency/nearby` endpoint. It utilizes the `BackendToolClient` to handle the HTTP requests to the backend. The chatbot logic would call the methods of this module to get the required information.
