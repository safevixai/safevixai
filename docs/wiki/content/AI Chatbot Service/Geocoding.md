# `geocoding.py` Module - SafeVixAI

## Overview

The `geocoding.py` module provides reverse geocoding functionality for the SafeVixAI chatbot service. It converts GPS coordinates (latitude and longitude) into human-readable addresses, utilizing Nominatim as the primary geocoding service and OpenCage as a fallback. This module is crucial for features like crash detection, SOS message generation, and context assembly within the chatbot.
