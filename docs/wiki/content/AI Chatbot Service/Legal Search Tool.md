# Legal Search Tool Module

## Overview

The `LegalSearchTool` module provides a specialized search functionality within the SafeVixAI platform, specifically designed to retrieve relevant legal information. It leverages a `Retriever` object to query a knowledge base and return results pertinent to legal queries, filtering based on the 'legal' scope. This module is a core component of the AI Chatbot Service, enabling the chatbot to provide informed responses related to legal aspects of road safety.
