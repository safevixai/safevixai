# Provider Router Module

## Overview

The `router.py` module is the central routing component for the SafeVixAI AI Chatbot Service. It intelligently selects the appropriate Large Language Model (LLM) provider from a pool of 11 options, based on factors such as user language, query intent, and provider availability. This module ensures optimal performance, cost-effectiveness, and fallback mechanisms for a robust and reliable chatbot experience.
