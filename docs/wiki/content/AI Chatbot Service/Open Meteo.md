# OpenMeteo Weather API Module

## Overview

The `open_meteo.py` module provides an interface to fetch weather data from the Open-Meteo API. This module is a crucial component of the SafeVixAI AI Chatbot Service, serving as the primary source for weather information used in assessing road safety risks. It retrieves current weather conditions, hourly precipitation probability, and visibility data, which are then used to calculate a risk multiplier.
