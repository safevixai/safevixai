The `EmergencyNumbers` component renders a navigation bar containing emergency phone numbers. It displays a list of pre-defined emergency services with their corresponding phone numbers, allowing users to quickly call them directly from the application. The component utilizes data from the `PRIMARY_EMERGENCY_BAR` constant to populate the navigation bar.
